# 1.获取jdk8下载链接

打开网址选择合适自己系统的jdk版本，点击下载 要注意选择.tar.gz的文件，如果不知道系统是32bit还是64bit,可以用uname -a查看 点击同意： 然后会自动跳转登录： 登录成功后会出现文件下载链接以及文件名称：

# 2.下载并解压

然后在centos中输入以下命令， 格式为：wget -O 文件名 网址 （文件名和网址是上图中复制粘贴过来的） 下载完成后解压文件：命令格式：tar zxvf 压缩包名称 -C 目的地路径

# 3.配置环境变量

解压后配置环境变量：输入vi /etc/profile进入vim编辑区 按i进入输入模式，然后一直按↓使光标移到文件末尾，添加以下配置信息：

#set java environment
JAVA_HOME=/usr/lib/jvm/jdk1.8.0_241
CLASSPATH=.:${JAVA_HOME}/jre/lib/rt.jar:${JAVA_HOME}/lib/dt.jar:${JAVA_HOME}/lib/tools.jar
PATH=$PATH:${JAVA_HOME}/bin
export JAVA_HOME CLASSPATH PATH
