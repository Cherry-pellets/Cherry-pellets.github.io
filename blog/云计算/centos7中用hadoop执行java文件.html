# 3.利用hadoop执行java包

（1）编辑自己的java文件，然后放到Hadoop目录下的javasource文件夹中 （2）编译java文件 格式：javac XXXX.java -d XXX （3）打包 格式：jar –cvf XXXXXX.jar –C XXX/ .

-c: 创建压缩文件
-v: 输出压缩详细情况
-f: 指定压缩文件
-m: 指定清单文件（manifest.mf是描述JAR内容的清单文件） 
-C dir：指定目录
